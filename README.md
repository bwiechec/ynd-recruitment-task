## Description

The goal of the task was to implement GitHub API and allow users to search for up to 5 GitHub users.

## Installation

To install the app run

```
yarn install
```

To start the app run

```
yarn run dev
```

To run Storybook use

```
yarn run storybook
```

To run tests use

```
yarn test
```
